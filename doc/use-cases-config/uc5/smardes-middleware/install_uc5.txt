Installation Notes for Use Case 5

1. Copy Config into smardes-middleware

Copy the whole content of uc5 into smardes-middleware-x.y.z\smardes-backend-services-x.y.z

2. Database Setup

a) Copy JDBC Driver for your database

... into smardes-backend-services-x.y.z\externaldb\lib

For example, for MS SQL Database this would be sqljdbc42.jar.

b) Modify externaldb/META-INF/persistence.xml

   Add the following entry:
	<persistence-unit name="uc5">
		<properties>
			<property name="hibernate.ejb.cfgfile" value="/hibernate_uc5.cfg.xml"/>
		</properties>
	</persistence-unit>

b) Configure externaldb/hibernate_uc5.cfg.xml

   Adjust DB Configuration, notably:
   hibernate.connection.url
   hibernate.connection.username
   hibernate.connection.password

c) Adjust config/queries/uc5.properties

   Might be needed to add a prefix to the table, e.g. schema name or similar.
